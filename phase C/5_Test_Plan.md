## 1. 測試總覽

### 1.1 測試階段

| 階段 | 名稱 | 內容 | 時間 |
|:---|:---|:---|:---|
| **T-1** | 單元測試 | 每個組件獨立測試 | 1 週 |
| **T-2** | 整合測試 | 子系統之間協同測試 | 1 週 |
| **T-3** | 全系統測試 | 完整通訊系統測試 | 1 週 |
| **T-4** | 太空環境測試 | 真實太空環境測試 | 1 週 |
| **總測試時間** | | | **~4 週** |


## 2. T-1：單元測試

### 2.1 T-1.1：負能量聚焦器測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證聚焦器能否將負能量聚焦到 1mm |
| **測試方法** | 用能量感測器測量焦點尺寸 |
| **測試步驟** | 1. 啟動聚焦器<br>2. 測量焦點尺寸<br>3. 記錄能量密度 |
| **合格標準** | 焦點 < 1mm，能量密度 > 目標值 |
| **失敗處理** | 重新校準聚焦器 |

### 2.2 T-1.2：場釋放閥測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證閥門開關時間同流量控制 |
| **測試方法** | 測量開關時間同流量 |
| **測試步驟** | 1. 測試打開時間<br>2. 測試關閉時間<br>3. 測試流量控制 |
| **合格標準** | 開關時間 < 1μs，流量精度 ±5% |
| **失敗處理** | 更換閥門 |

### 2.3 T-1.3：激光編碼器測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證數據編碼同解碼準確性 |
| **測試方法** | 發送測試數據，比對輸入輸出 |
| **合格標準** | 100% 準確 |
| **失敗處理** | 重新校準編碼器 |

### 2.4 T-1.4：量子干涉儀感測器測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證感測器靈敏度 |
| **測試方法** | 測量已知信號源 |
| **合格標準** | 靈敏度 < 10⁻²⁰ 應變 |
| **失敗處理** | 更換感測器 |


## 3. T-2：整合測試

### 3.1 T-2.1：聚焦器 → 閥門 整合測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證聚焦器到閥門嘅能量傳輸 |
| **測試方法** | 監測能量傳輸效率 |
| **合格標準** | 傳輸效率 > 99% |
| **失敗處理** | 檢查管道連接 |

### 3.2 T-2.2：閥門 → 編碼器 整合測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證數據編碼同曲率泡形成嘅同步 |
| **測試方法** | 發射測試數據，監測曲率泡 |
| **合格標準** | 數據正確編碼到曲率泡 |
| **失敗處理** | 檢查同步時序 |

### 3.3 T-2.3：發射器 → 接收陣列 整合測試

| 項目 | 內容 |
|:---|:---|
| **測試目標** | 驗證發射器同接收陣列之間嘅通訊 |
| **測試方法** | 發射測試信號，監測接收 |
| **合格標準** | 成功接收並解碼 |
| **失敗處理** | 檢查指向同校準 |
