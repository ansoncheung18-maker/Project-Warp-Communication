## 3. 核心技術

### 3.1 信息載體：1mm 曲率泡

| 參數 | 數值 | 備註 |
|:---|:---|:---|
| 直徑 | 1 mm | 極微型曲率泡 |
| 速度 | 20,000c – 500,000c | 分階段目標 |
| 負能量需求 | ~10⁻⁶ – 10⁵ J | 視乎速度而定 |
| 信息編碼 | 激光脈衝編碼 | 將數位數據轉化為光信號 |

### 3.2 發射系統

| 組件 | 功能 | 技術來源 |
|:---|:---|:---|
| **微型 Casimir 陣列** | 產生負能量 | Project 8 |
| **負能量聚焦器** | 將負能量聚焦到 1mm 焦點 | 新設計 |
| **超導儲能環** | 累積負能量，瞬時釋放 | Project 4 |
| **場釋放閥** | 控制負能量釋放速率 | Project 8 |

### 3.3 減速與接收機制

為確保信息載體（1mm 曲率泡）能夠被準確接收，需要喺到達目的地星系前進行「主動減速」。

#### 減速策略

| 階段 | 位置 | 速度變化 | 說明 |
|:---|:---|:---|:---|
| 巡航 | 距離目的地 > 5 光年 | 保持巡航速度 | 極速飛行 |
| **開始減速** | 距離目的地 1 光年 | **50,000c → 1,000c** | 減少負能量供應 |
| **繼續減速** | 距離目的地 0.1 光年 | **1,000c → 10c** | 繼續減速 |
| **進場** | 距離目的地 0.01 光年 | **10c → 1c** | 接近亞光速 |
| **到達** | 目的地軌道 | **1c → 0** | 接近靜止，準備接收 |

#### 能量回收

減速過程中釋放嘅能量可以俾接收站回收，作為額外嘅能量來源。

### 3.4 接收系統

| 組件 | 功能 | 技術來源 |
|:---|:---|:---|
| **量子干涉儀陣列** | 檢測 1mm 曲率泡嘅時空扭曲訊號 | Project 8 |
| **信號解碼器** | 將光信號轉換為數位數據 | 新設計 |
| **通訊協議** | 定義數據包格式同錯誤校正 | 新設計 |

### 3.5 穩定系統

| 組件 | 功能 | 技術來源 |
|:---|:---|:---|
| **主動邊界穩定系統** | 實時補償曲率泡邊界嘅時空壓力 | Project 8 (AI 控制) |
| **量子干涉儀感測器** | 監測邊界穩定性 | Project 8 |
| **AI 預測控制** | 預測並補償不穩定趨勢 | Project 6 (Fusion Health) |

## 6. 主要挑戰同緩解措施

| 挑戰 | 緩解措施 |
|:---|:---|
| 1mm 曲率泡穩定性 | 主動邊界穩定系統 + AI 預測控制 |
| 極高速度下嘅能量聚焦 | 超導儲能環 + 場釋放閥瞬時釋放 |
| 超長距離精確導向 | 量子干涉儀陣列 + AI 實時微調 |
| 信號衰減同干擾 | 激光脈衝編碼 + 錯誤校正協議 |
| 接收站檢測靈敏度 | 大型量子干涉儀陣列 + 編碼脈衝簽名 |
| **曲率泡到達速度過快** | **預先減速機制（1 光年外開始減速）** |
